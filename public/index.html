Audio Visualizer

How to Use

Control the visualizer using the control panel in the top-right of the web page. You can:

Control Function
Play/Pause Toggle music playback.
Toggle Text Toggle this overlay text on and off.
Volume Change the volume multiplier.
Spokes Change the number of volume-sensitive spokes, distributed evenly across all frequencies.
Shape Choose the shape of the visualizer.
Colors > Gradient Disable or change the orientation of the gradient.
Colors > Shape Change the colors of the visualizer foreground object.
Colors > Background 1 Change the primary background color.
Colors > Background 2 Change the secondary background color. Ignored if gradient is set to "Solid".

You can also hide/show the control panel using the "h" key.
