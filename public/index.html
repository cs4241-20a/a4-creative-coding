Conway's Game of Life Help

  • Resizing the window
    If you resize the window, the game size will automatically adjust to the new window size. This will restart the game.

  • Pause
    While checked, the game will be paused and no generations will occur.

  • Enable wrapping
    While checked, the borders of the screen will wrap to the other side (i.e. the game will run in a toroidal world).

  • Live cell color
    This sets the color of living cells. By default, living cells are black.

  • Generations per second
    This sets the number of generations that will occur each second while the game is unpaused. For high values, your computer might not be able to keep up and you may not actually get the number of generations per second that the game is set to. When this happens, red text that says "Can't keep up!" will appear in the top left.

  • Render scale
    This changes how many pixels each cell takes up on the screen. Larger values increase performance. Changing this value will restart the game.

  • Flip alive/dead
    This button flips the living/dead state of every cell.

  • Randomize game
    This button randomly sets the alive state of each cell, restarting the game.

  • View instructions
    This button takes you to this page.
